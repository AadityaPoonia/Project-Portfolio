# Classic ML Notebooks

A collection of foundational machine learning notebooks covering classification, regression, and anomaly detection on well-known datasets. Completed during a data science internship with **CodSoft**.

## Projects

### 🚢 Titanic Survival Prediction
Predicts passenger survival on the Titanic dataset using classification models, with feature engineering on passenger class, age, fare, and family structure.
[`titanic-dataset.ipynb`](./titanic-dataset.ipynb)

### 🎬 Movie Rating Prediction
Analyses the factors influencing IMDB movie ratings and builds predictive models to estimate a film's rating from its attributes.
[`movie-rating-prediction.ipynb`](./movie-rating-prediction.ipynb)

### 📈 Sales Prediction
Forecasts sales from advertising spend using linear regression to inform marketing allocation.
[`sales-prediction-simple-linear-regression.ipynb`](./sales-prediction-simple-linear-regression.ipynb)

### 💳 Credit Card Fraud Detection
Detects fraudulent transactions with a classification model tuned for imbalanced data, prioritising recall on the fraud class.
[`credit-card-fraud-detection.ipynb`](./credit-card-fraud-detection.ipynb)

### 🌸 Iris Flower Classification
Classifies iris species from sepal and petal measurements — a compact introduction to multiclass classification.
[`iris-flower-classification.ipynb`](./iris-flower-classification.ipynb)

## Tech Stack

Python, pandas, NumPy, scikit-learn, Matplotlib, Seaborn, Jupyter

## Acknowledgement

These projects were completed as part of a data science internship with CodSoft, which provided hands-on experience across the machine learning workflow — from data cleaning and feature engineering to model evaluation.
